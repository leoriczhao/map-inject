5. 调试输出

三重输出机制（dbg_printf, lua_bootstrap.cpp:27）：

  1. DebugView — OutputDebugStringA，使用 tools\Dbgview.exe 实时查看
  2. 控制台 — AllocConsole + WriteConsoleA，游戏运行时可见独立控制台窗口
  3. 日志文件 — japi-debug.log，带时间戳的持久化日志
void dbg_printf(const char* fmt, ...) {
    char buf[1024];
    int n = vsnprintf(buf, sizeof(buf), fmt, va);
    OutputDebugStringA(buf);              // → DebugView
    WriteConsoleA(stdout, buf, n, ...);   // → 控制台
    if (g_log) { fwrite(buf, 1, n, g_log); fflush(g_log); }  // → 日志文件
}
