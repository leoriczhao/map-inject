1.3 场景/加载分析系列

2. Callback Exploit 代码

tools/callback

1057 行 JASS 代码（AI 脚本格式），在 War3 加载时由 StartCampaignAI 触发。执行：版本检测 → UnlockMemory → 分配 RWX 内存 → ExportFileFromMpq → LoadLibrary(japi.dll) → GetProcAddress("call") → CallCdecl(call, pExportFromMpq)。
文件: tools/callback (1,057 行), tools/callback_ydwe.j (YDWE 版本参考)

reference/builtin-japi/callback

内置 JAPI 的原始 callback 代码（908 行）。与我们的版本几乎相同，但 main() 末尾调用 "Initialize" 导出（而非 "call"）并传递 0（而非 pExportFromMpq）。
文件: reference/builtin-japi/callback (908 行)

3. pjass — JASS 语法检查器

tools/pjass-src/

完整的 pjass 源码（C 语言实现的 JASS 词法/语法分析器，YACC 语法 + Flex 词法），约 120 个测试用例覆盖 11 个测试类别。用于验证 callback 代码和 war3map.j 的语法正确性。
文件: tools/pjass-src/ (C 源码 + YACC 语法 + Flex 词法 + 120 测试), tools/pjass.exe, tools/pjass-std.exe

4. w3x2lni — MPQ 地图转换器

tools/w3x2lni/

全功能 Warcraft III 地图格式转换器，支持 w3x（MPQ）和 lni（目录格式）之间的双向转换。Lua 5.4 驱动，包含 GUI 和 CLI 界面、完整的 Warcraft III 游戏数据预置（enUS-1.27.1 / zhCN-1.24.4 / zhCN-1.32.8）、SLK 回端/前端、地图构建器、FFI 绑定。build.ps1 使用其 CLI 模式解包/打包 demo.w3x。
文件: tools/w3x2lni/bin/w3x2lni-lua.exe, tools/w3x2lni/script/ (100+ Lua 模块)

9. inject_ujapi_wrapper.ps1

reference/ujapi/inject_ujapi_wrapper.ps1

37 行 PowerShell 脚本。将 ujapi_wrapper.j（基于 hashtable 的 UjAPI 分发包装器）注入到 war3map.j 的 //* Main Initialization 标记前。支持增量注入（检测已存在的 wrapper 并替换）。
